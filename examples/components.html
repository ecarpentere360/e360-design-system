Usage Guidelines

Button Guidelines

  • Use Primary buttons for main actions (Submit, Save, Continue)
  • Use Secondary buttons for supporting actions
  • Use Outline buttons for less prominent actions
  • Always provide clear, action-oriented button text

Card Guidelines

  • Use cards to group related information
  • Keep card content concise and focused
  • Use consistent spacing and typography within cards
  • Cards work well in grid layouts for content organization

Form Guidelines

  • Use clear, descriptive labels and placeholders
  • Provide immediate feedback for validation errors
  • Group related form fields together
  • Use appropriate input types for better mobile experience
