Paragraph Typography

Used for body text, descriptions, and content.

Paragraph Large - This is used for important body text and lead paragraphs. It provides emphasis and draws attention to key information while maintaining excellent readability.

Font Size: 20px
Line Height: 28px
Font Weight: 500 (Medium)
Letter Spacing: 0px
Paragraph Spacing: 20px
Usage: Lead text, important descriptions, introductory paragraphs, call-to-action text

Paragraph Medium - This is the standard body text used throughout the design system. It provides optimal readability for most content while maintaining consistency across all interfaces and components.

Font Size: 16px
Line Height: 24px
Font Weight: 400 (Regular)
Letter Spacing: 0px
Paragraph Spacing: 16px
Usage: Standard body text, descriptions, form labels, general content

Paragraph Small - This is used for captions, footnotes, and secondary text. It provides additional information without overwhelming the main content.

Font Size: 13px
Line Height: 19.5px
Font Weight: 400 (Regular)
Letter Spacing: 0px
Paragraph Spacing: 16px
Usage: Captions, footnotes, metadata, fine print, secondary information

Implementation Examples

CSS Classes

<h1 class="display-1">Hero Title</h1> <h2 class="display-2">Page Title</h2> <p class="paragraph-lg">Important content</p> <p class="paragraph-md">Standard content</p> <p class="paragraph-sm">Caption text</p>

.custom-heading { font-family: var(--font-family-primary); font-size: var(--display-3-size); font-weight: var(--display-3-weight); line-height: var(--display-3-line-height); letter-spacing: var(--display-3-spacing); }

.hero-title { font-family: 'DM Sans', sans-serif; font-size: 96px; font-weight: 400; line-height: 105.6px; letter-spacing: -1.92px; }

Accessibility Guidelines

Contrast Requirements

  • Normal text: Minimum 4.5:1 contrast ratio
  • Large text (18px+): Minimum 3:1 contrast ratio
  • Bold text (14px+): Minimum 3:1 contrast ratio

Line Height

  • All typography styles include generous line heights for readability
  • Line heights are optimized for each font size
  • Minimum 1.5x line height for body text

Font Loading

  • DM Sans is loaded from Google Fonts with font-display: swap
  • Fallback fonts ensure text remains readable during loading
  • System fonts used as fallbacks for optimal performance
